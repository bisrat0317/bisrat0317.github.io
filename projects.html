ICT Support and Management Platform

A customizable ICT support and management platform developed as my final university project. It streamlines the process of submitting, tracking, and managing IT support requests across different organizations.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL

Project Features & Goals

  • Submit and track IT support requests via web interface
  • Role-based access for users, approvers, managers, technicians, and admins
  • System customization for different organizational needs
  • Customer satisfaction feedback collection
  • Automated notifications and status updates

My Role & Contributions

  • System analysis and requirements gathering from stakeholders
  • Customizing and coding new features using PHP, JavaScript, and MySQL
  • System testing and bug fixing based on user feedback
  • Database design and integration
  • Collaborated closely with mentors and end-users to ensure usability

Challenges & Solutions

  • Understanding legacy system code and adapting new features - solved through code walkthroughs and consistent learning
  • Gathering requirements from different user roles with varying needs - solved via interviews and iterative feedback
  • Balancing user-friendly design with technical limitations - addressed by continuous testing and UI adjustments
